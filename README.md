# Material_Storage (MS)

Minecraft Paper / Purpur 用  
単一アイテム超圧縮ストレージプラグイン

対応：
- Minecraft 1.21.11
- Java 21
- Paper / Purpur

---

# 特徴

- 単一アイテムを超大量収納
- 完全一致NBT対応
- 自動回収
- コンテナ一括収納
- ホッパー搬送対応
- ホッパーフィルター対応
- ストレージ間搬送対応
- Lore / 表示名自動更新
- 擬似グロウ表示
- 安全なPDC保存
- Vanillaライク設計
